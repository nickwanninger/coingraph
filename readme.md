# coingraph
## A simple crypto currency graph for the terminal

![preview](https://i.imgur.com/sxJcYFQ.jpg)
---
## install

`npm install -g coingraph`

---

## Options

- `--help` for help
- `-h` hour timeframe
- `-d` day timeframe
- `-w` week timeframe
- `-m` month timeframe
- `-y` year timeframe
- `-a` show value of the coin over its entire life
- `-c [name]` name of coin (btc, eth, ltc)
